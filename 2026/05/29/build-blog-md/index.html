作为计算机大类的学生，我们都很渴望拥有一个属于自己的博客，记录自己在学习计算机科学过程的点点滴滴。我个人认为一种比较简单的方式就是使用Hexo + GitHub Pages 搭建。如果你使用的是 Arch Linux（和我一样），那事情就变得更简单也更高能了！作为 Arch 用户，你一定习惯了滚动更新和高效的包管理器。在 Arch 上配置 Node.js 环境并搭建 Hexo，比在 Windows 或 Mac 上还要丝滑。
